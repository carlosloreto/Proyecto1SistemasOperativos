El programa memory les permitira calcular la memoria m[axima que esta utilizando un programa.
Para compilar el programa emplee el siguiente comando:

$ gcc -o memory memory.c

Para usar luego el comando emplee la siguiente sintaxis

$ memory <programa> <parametros>

Donde programa es el nombre del programa al que desea medirle cual es la cantidad de memoria que usa, y parametros son los parametros de operacion que debe usar el programa <programa>.
